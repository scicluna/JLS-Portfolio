Server Crawl

Server Crawl was my first attempt to build a full-stack web application using Express to maintain the routing and a basic JSON database API to feed the game with various bits of information. This project was a great success and really brought together front-end with back-end for me heavily utilizing fetch requests to grab information from my API. It also includes advanced javascript logic for "dungeon" generation and advanced CSS for animations. I also heavily utilized classes and object oriented programing to maintain and update the various gameplay objects that the player interacts with.
